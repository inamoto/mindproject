# Mindproject

Mindprojectはプロジェクトマネジメントのためのクラウドサービスとして開発しています。

Mindmapで全体像をは把握しながら作業項目を洗い出し、ガントチャートでスケジュールと担当者を決定して、進捗を管理するといった用途でのご利用を想定しています。

ただ、単純にWebで使えるMindmapソフト、ガントチャートソフトとしてご利用いただいてもかまいません。

## 開発環境
- php 5.6.30
- mysql
- laravel 5.4

## ソースに含まれないもの
- .env 
- ventorディレクトリ以下のファイル
